Color Schema

The first color that I want to use is red like Nintendo uses in it's logo #fe0016 for the nav bar and footer. I do want to use a slightly darker shade.

I want to use dark gray for the background and white for text. This will make it easier on the eyes

I want to use red as an accent color because red and black look cool together.

Typography

I decided to use the "Georgia" font. I chose it because it is a common system font that is clear and nice.
